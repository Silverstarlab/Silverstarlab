## About
- 실전 환경에서 **공매도 봇을 실테스팅**하며, **크립토 시장 분석/전략 검증**까지 확장 중입니다.
- 목표는 “아이디어”가 아니라, **데이터 수집 → 검증 → 실행/모니터링**까지 end-to-end로 돌아가는 퀀트 시스템을 만드는 것입니다.
- 과거에 **나스닥 공매도 파생시장 흐름을 예측하는 알고리즘을 개발**해 실전에서 유의미한 성과를 낸 전적이 있습니다.

## Focus Areas
- **Market Data Pipeline**: 거래소 API 연동, 스케줄링, 저장 구조, 정제/품질 관리
- **Signal/Strategy Layer**: 백테스트/검증, 시그널 생성, 리스크 제약
- **Execution & Risk**: 포지션/주문/로그/알림, 운영 안정성(예외/장애 대응)
- **Ops & Observability**: 배포 자동화, 모니터링, 재현 가능한 실험 환경
